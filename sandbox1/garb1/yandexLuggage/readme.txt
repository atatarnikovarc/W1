==Предпусковые условия
1. MySql запущен
2. база с именем app.properties-->db.name создана
3. java установлена на хосте запуска
4. установлена переменная окружения YA_ROOT, указывающая на корень фреймворка(rant.sh\rant.bat файл)

==Замечания
1. все параметры управляются через файл $YA_ROOT\config\app.properties
2. файлы данных находятся в папке data - flights.xml, test-data-sql.sql

==Запуск из командной строки

./rant.sh(или rant.bat) (из папки фреймворка)

==Запуск из eclipse

1. прописать в пути (build path) директорию с файлом app.properties
2. запустить класс RunLuggage

==Остановить приложение
Либо ввести имя - "hihi"
Либо ctrl+c
