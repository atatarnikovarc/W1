Эта утилита используется для генерации xml файла архива мелодий, а также директории,
которая содержит файлы, соответствующие архиву

===Предварительные требования
Установленная java 1.6  и выше

===Порядок работы

1. удалить папку .svn из папки runxml\output\1
2. Открыть файл run.bat из текущей директории(где находится сам readme.txt) на редактирование
3. Выбрать нужные параметры для генерации
4. Открыть на редактирование файл runxml\data\1253.xml и убедиться, что значения тегов
корректны для аггрегатора, для которого осуществляется заливка мелодий
4.1 Если есть потребность сделать архив для CPA аггрегатора, то нужно использовать
этот файл runxml\data\cpa.xml в качестве значения параметра xmlFilename
5. Выполнить run.bat из п.1
6. Открыть файл, имя которого находится по пути outputXmlFileName (см. run.bat из п.1) и заменить
строчку <!--DOCTYPE Inventory SYSTEM "inventory1.dtd"--> на строчку
        <!DOCTYPE Inventory SYSTEM "inventory1.dtd">
7. сделать zip-архив на основе папки путь к которой описывается в параметре copyPath
(не забыть - что все файлы должны быть в архиве без папки - т.е. при открытии архива - 
должны сразу отображаться все файлы)

Теперь у вас есть архив мелодий, определяющийся двумя файлами:
outputXmlFileName и имени архива из п.7
