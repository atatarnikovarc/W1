Этот фреймфорк выполняет авто тесты для веб-апи функций продукта x-tones.
Описание файлов:
-run.bat - запуск тестов под windows
-./config/app.properties - общие настройки фреймворка - база, URL приложения для тестов и т.п.
-%RUNAPI%\result\html\index.html - файл отчета выполнения тестов


Условия для запуска тестов:

1. Переменная среды RUNAPI должна быть установлена в корень папки фреймворка


Предположения:

1. Пароли для всех админов - пустые
2. Телефонные номера должны позволять добавлять админов(ёмкость среди corp_representatives)
3. В системе должна быть хотя бы одна корпорация с хотя бы одним администратором
4. Не обрабатываются тесты, когда база не доступна
